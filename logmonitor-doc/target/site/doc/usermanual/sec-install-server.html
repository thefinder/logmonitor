4.2. 安装服务器

按如下步骤完成配置：

  1. 将下载的logmonitor-server.zip解压到指定目录，例如：d:/server/logmonitor-server。

  2. 执行bin\InstallApp.bat安装为Windows服务，名为“LogMonitor Server”。

  3. 打开conf/logmonitor-server.properties，做如下配置：

    # 服务器端口
logmonitor.server.port=15880

这里15880是默认端口，可以修改。

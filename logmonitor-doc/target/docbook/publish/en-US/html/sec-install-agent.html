4.3. 安装客户机代理

按如下步骤完成配置：

  1. 将下载的logmonitor-agent.zip解压到指定目录，例如：d:/server/logmonitor-agent。

  2. 执行bin\InstallApp.bat安装为Windows服务，名为“LogMonitor Agent”。

  3. 打开conf/logmonitor.properties，做如下配置：

    # 日志服务器地址
logmonitor.server=192.168.1.2:15880
# 本机IP
logmonitor.host=192.168.1.10
# 日志扫描线程数
logmonitor.threadcount=1
# 日志扫描时间间隔（单位：秒）
logmonitor.scanseconds=30
# 心跳时间间隔（单位：秒）
logmonitor.heartbeatseconds=30

这里logmonitor.server需要配成前面配置的服务器地址，host是客户机的IP。
